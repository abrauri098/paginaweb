Atributos Comunes de Validación HTML5

Atributo Descripción Ejemplo
required Campo obligatorio <input required>
min/max Valores mínimo y máximo <input type="number" min="0" max="100">
pattern Expresión regular de validación <input pattern="[A-Z]{3}">
minlength/maxlength Longitud mínima y máxima de texto <input minlength="5" maxlength="20">
step Incremento permitido para números <input type="number" step="0.5">
placeholder Texto de sugerencia <input placeholder="Ingrese texto...">
disabled Desactiva el campo <input disabled>
readonly Solo lectura <input readonly>
